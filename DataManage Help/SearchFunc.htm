Search Function

this function enables you to search your recordset for any combination with Exact or Soft seek
this mean you can search for "Computer" and find any field with this word exactly , or you can search for "Com" and find any field contain this letters like "Commerce" , "Welcome"

Syntax:

Function Search(ByRef dm_Form As System.Windows.Forms.Form, ByRef dm_MasterTable As ADODB.Recordset, Optional ByRef dm_Grid As AxMSDataGridLib.AxDataGrid = Nothing, Optional ByRef dm_DetailTable As ADODB.Recordset = Nothing)

Example:

Dim oMaster As New ADODB.Recordset()
Dim oDetails As New ADODB.Recordset()

DM.Search(Me, oMaster, MyGrid, oDetails)
