DeleteRecord

this function delete current record and populate your form and its grid with previous record data

Syntax:

Function DeleteRecord(ByRef dm_Form As System.Windows.Forms.Form, ByRef dm_MasterTable As ADODB.Recordset, Optional ByRef dm_Grid As AxMSDataGridLib.AxDataGrid = Nothing, Optional ByRef dm_DetailTable As ADODB.Recordset = Nothing)

Example:

DM.DeleteRecord(Me, oServer)
