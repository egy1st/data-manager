Overview

Data Manger
DC Data Manger lets you navigate and manipulate your database records any way you want. This program is essentially a beefed-up version of DC Binding Recordset , Forget about coding the hard way – with Data Manger, you can add, save, delete, or edit data using only simple codes.

Easy integration
Like its predecessor, DC Data Manger assimilates flawlessly into your codes – you don’t have to launch it every time you open your program. It works with Visual Basic, Visual C++, and other such COM-based environments.

Database manipulation
DC Data Manger also allows setting of database table relationships and establishment of search functionalities. It also lets you implement data entry validation and format data by defining decimal places for numeric entries or letter cases for character entries. This way, you don’t have to manually change each entry.

Interactive buttons
DC Data Manger is integrated with Image Button, which allows you to add interactive images to your program. You can set your button images to change according to the user’s mouse movements and positions. 

Built-in translation tools
DC Data Manger has a built-in form translator and a form flipper (both of which are also available separately form EgyFirst). These applications allow DC Data Manger to support multi-language applications and automatically populate form labels with predefined translations. It also flips your form 180 degrees to support Eastern language characters.

Easy to use
You do not need to be an advanced programmer in order to install and fully use Binding Recordset. Its setup program automatically registers the DC_DataManger10.dll file on your system. To integrate it into your project, simply select 'add reference' from the project menu.  Click the 'browse' button to locate all of these  DLL :

1-  your DC_BindingRecordset10.DLL file, which by default is located in c:\windows\system32.
2-  AxInterop.MSDataGridLib.dll  which by default is located in c:\windows\system32.
3-  Interop.MSDataGridLib.dll  which by default is located in c:\windows\system32.
4- adodb.dll  which by default is located in c:\windows\system32.
