HoldSaving

this function stop saving data with current record  until you call RelaseSaving
this function is usuful when you want to add additional saving code , or to get the mose contol on saving data

Syntax:

Public Sub HoldSaving(ByVal Mode As Boolean)

Example:

If Not DM.ValidateForm(Me) Then Exit Sub

DM.HoldSaving(True) // stop saving date untill calling for ReleaseSaving

DM.SaveData(Me, oServer) //

oServer.Fields("Items2").Value = Me.PreItems.Text  // additional code handling by you

oServer.Fields("Items2").Value += Me.Items.Text    // additional code handling by you

oServer.Fields("Items2").Value += Me.PostItems.Text  // additional code handling by you

DM.ReleaseSaving(True) save data
