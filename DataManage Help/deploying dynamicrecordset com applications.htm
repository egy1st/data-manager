Deploying DC.DataManger Applications

The only file needs to be distributed with DC.DatManger COM applications is DC_DataManger30.dll
this file should be copied to the WinSystem directory which by default is C:\Windows\System32

Note :
If you deploy a translated application with your local language , you must deploy DCDM10_Lang.dll which located in your system directory which is by default C:\Windows\System32\  and place it in end user system directory
