Including DC.DataManger

To include DC.DataManger in your project

  1. From Project menu select add reference

  2. Push Browse button to locate your DC_DataManger30.DLL file which by default located in c:\windows\system32  , now the specific DLL included in your references

  3. Be sure that Micorsoft DataGrid Control Vresion 6.0 (OLEDB) is included in your Toolbox, if not then right click Toolbox

  4. From shortcut menu select customize Toolbox

  5. From customize Toolbox Dialog ,scroll down and check Micorsoft DataGrid Control Version 6.0 (OLEDB)
